# MCP App Hackathon

MCP App built for the Claude Code London 03 hackathon (Feb 20, 2026).

## Setup

```bash
pnpm install
pnpm dev
```

## What is this?

An MCP App built with [Skybridge](https://github.com/alpic-ai/skybridge) that solves an everyday at-work issue.
